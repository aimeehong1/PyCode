LISTS

Loops are a great way to run a command multiple times in a simple and efficient way.
There are three main types of loops in python. Scroll down to read more about them!

While Loop

A while loop runs as long as its input statement is true

Example:

i = 0
while: i < 5:
     print("hello ")
     i+=1

Output:

hello hello hello hello hello

For Loop

A for loop runs for a defined number of times

Example:

for(int i = 0; i < 5; i ++)
print("hello ")

Output:

hello hello hello hello hello

Nested Loop

A nested loop operates by executing one loop inside another loop. This can be a combination of for/while loops

Example:

for(int i = 0; i < 2; i++)
     for(int i = 0; i < 2; i++)
          print("hello ")

Note: Inner Loop runs twice. Outer loop runs twice. So total run-time of print is 2x2 = 4 times.

Output:

hello hello hello hello
