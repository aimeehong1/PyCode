Working with Lists

What are lists?

Lists are a structure where you can add and remove items of different data types. Imagine the types of lists we ordinarily make. We make grocery lists full of items (strings!).

Grocery List
  • A dozen eggs 🥚🥚
  • Milk 🥛
  • Bacon 🥓
list = ["A dozen eggs", "Milk", "Bacon"]

We also have lists full of numbers.

Number List

  • 123
  • 234
  • 345
list = [123, 234, 345]

Or a mixture of both!

Restaurant Menu

  • Rib Steak ...... $18.99 🥩
  • Clams (x8) ..... $16.99
  • Lobster ...... $22.00 🦞
list = ["Rib Steak", 18.99, "Clams", 16.99, "Lobster", 22.00]

How do I code a list?

Creating a list is simple! You can use the list() function or brackets []!
- sample_list = list()
- sample_list = []

But we don't just want an empty list! So we need to add something to our newly created list. This will require anoter function, the append() function which will add our item to the end of a list.
For example, if we write sample_list.append(1) , the number one will be added to our empty list.

This will be our output:
sample_list = [1]

If we append again, using sample_list.append(2) our list will be the foloowing:
sample_list = [1, 2]

And one final append, sample_list.append(3) will add the number 3 to the end of our list and we end up with
sample_list = [1, 2, 3]
